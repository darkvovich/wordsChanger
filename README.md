# wordsChanger
jQuery plugin to change words with awesome letter effect!

### Demo: [view here](http://darkvovich.github.io/wordsChanger.html)

---

### Usage:
#### HTML:
```
<div id="word" data-words="word, another, many, words">
	Fixed <i>word</i>
</div>
```

#### JS:
Script required jQuery.
```
$('#word').wordsChanger();
```
